
## 昆仑谣H5 - 尾声场景(s11)手势交互修复

**问题**: s11尾声场景的手势完全没有专属效果。isEndActive()函数虽已定义，但三处手势分发(finger/fist/open)都未引用它，所有手势fall through到通用punchEffect或切场景。

**修复**:
1. 新增  - 比1手势: 墨粒子散落+诗词文字浮现+涟漪 (MAX_END_POEMS=5限流)
2. 新增  - 出拳手势: 金色墨爆粒子24个+中心光晕扩散+水花涟漪
3. 新增  - 挥手手势: 柔和金墨韵扩散+渐隐文字(文脉/不死/永存/始)
4. 三处手势分发加入  分支:
   - finger: isEndActive → endInkScatter (墨迹散落成诗)
   - fist: isEndActive → endBurst (金墨绽放)
   - open: isEndActive → endEcho (尾声回响), 不再触发切场景
5. 文件行数: 3995 → 4092
